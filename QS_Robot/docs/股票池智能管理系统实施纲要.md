## 1. 系统概述

### 1.1 系统定位
股票池智能管理系统是 QS Robot 量化交易平台的核心组件，负责实现股票的智能筛选、分类、动态管理和策略匹配，支撑自动化交易决策流程。

### 1.2 核心目标
- **智能化筛选**: 基于多维度分析自动筛选优质股票
- **动态管理**: 实现股票池的自动更新和生命周期管理
- **策略匹配**: 智能匹配股票与交易策略
- **风险控制**: 金融级风险评估和合规检查
- **自动化决策**: 端到端自动化流程，减少人工干预

### 1.3 设计原则
| 原则 | 说明 |
|------|------|
| **金融级标准** | 满足99.99%可用性、金融级加密、合规审计 |
| **多维度决策** | 预实盘模拟+技术指标+风险评估+专家规则 |
| **可扩展性** | 模块化设计，支持策略和数据源扩展 |
| **可解释性** | 透明的决策过程和评分 breakdown |
| **稳健性** | 多重校验和风险控制机制 |

class SmartStockFilter:
    """智能股票筛选引擎"""
    
    def filter(self, universe):

### 3.3 策略-股票匹配引擎

#### 3.3.1 匹配算法

```python
class StrategyStockMatcher:
    """策略与股票智能匹配"""
    
    def _calculate_match_score(self, features, strategy):
        score = 0
        
        # 波动率匹配（25%）
        if features['volatility'] == strategy['volatility_profile']:
            score += 25
        
        # 趋势强度匹配（25%）
        trend_match = 1 - abs(features['trend_strength'] - strategy['ideal_trend'])
        score += trend_match * 25
        
        # 流动性匹配（25%）
        if features['liquidity'] >= strategy['min_liquidity']:
            score += 25
        
        # 历史表现匹配（25%）
        if features['historical_return'] >= strategy['min_return']:
            score += 25
        
        return score
```

### 4.3 综合评分算法

```python
class CompositeScorer:
    """综合评分器"""
    
    def calculate(self, evaluations):
        weights = {
            'simulation': 0.40,
            'technical': 0.25,
            'risk': 0.20,
            'expert': 0.10,
            'compliance': 0.05
        }
        
        # 合规检查是门槛条件
        if evaluations['compliance']['passed'] == False:
            return {'score': 0, 'reason': '合规检查未通过'}
        
        # 计算加权得分
        total_score = 0
        for key, weight in weights.items():
            total_score += evaluations[key]['score'] * weight
        
        # 额外惩罚项
        if evaluations['risk'] < 60:
            total_score = max(0, total_score - 20)
        
        # 额外奖励项
        if evaluations['simulation'] >= 80:
            total_score = min(100, total_score + 5)
        
        return {
            'score': round(total_score, 2),
            'passed': total_score >= 70
        }
```

### 4.4 表决结果判定

| 综合得分 | 结果 | 处理方式 |
|----------|------|----------|
| ≥ 85 | 优秀 | 直接进入实盘池 |
| 70 - 84 | 良好 | 进入实盘池，加强监控 |
| 60 - 69 | 观察 | 进入观察池，定期重评 |
| < 60 | 拒绝 | 淘汰或重新筛选 |
