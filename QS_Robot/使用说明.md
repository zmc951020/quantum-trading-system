
# QS Robot 使用说明

## 快速启动服务

### 方法 1：在 PowerShell 中手动启动（推荐）

打开 PowerShell，依次执行：

```powershell
# 进入目录
cd d:\Gupiao\升级vscode\QS_Robot

# 启动服务
python start_ui_simple.py
```

然后在浏览器中访问：
**http://127.0.0.1:5001**

---

## 功能说明

### 界面操作
1. **点击右下角的悬浮球** → 打开侧边聊天窗口
2. 在输入框中输入问题，点击“发送”
3. 也可以点击快捷按钮快速发送预设命令

### 状态指示器
- **LLM**: Ollama 模型服务连接状态
- **系统**: Aurora 量化系统连接状态

### 快捷按钮
- **系统状态**: 查看系统当前状态
- **策略列表**: 查看可用策略列表
- **优化策略**: 优化策略参数
- **健康检查**: 执行健康检查
