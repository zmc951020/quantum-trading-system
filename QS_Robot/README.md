
# QS Robot - 量化系统神经中枢

## 项目简介

QS Robot 是 Aurora量化交易系统的神经中枢与智能助手平台，提供：

- **系统集成层**：便捷地使用策略、优化器、回测、风控、监控等模块
- **深度优化引擎**：辅助优化策略参数与系统配置
- **扩展插件体系**：支持自定义插件、工具、数据源和LLM提供者
- **智能助手UI**：悬浮球 + 侧边窗，支持语音交互

---

## 目录结构

```
QS_Robot/
├── __init__.py          # 包初始化
├── llm_manager.py       # LLM管理器
├── README.md            # 项目说明
├── config/              # 配置管理
│   ├── __init__.py
│   └── config.py        # 配置管理器
├── extensions/          # 扩展接口（插件化架构）
│   ├── llm_providers/   # LLM提供者（Ollama等）
│   ├── tools/           # 工具注册
│   ├── hooks/           # 钩子机制
│   ├── capabilities/    # 能力扩展
│   └── data_sources/    # 数据源
├── plugins/             # 插件目录
│   ├── official/        # 官方插件
│   ├── community/       # 社区插件
│   └── local/           # 本地自定义插件
├── memory/              # 记忆系统
├── docs/                # 文档目录
└── ui/                  # 用户界面
```

## 快速开始

### 1. 基础环境

要求：
- Python 3.8+
- Ollama服务（可选，但推荐）

### 2. 配置系统

首次运行时，会自动创建配置文件 `config.json`。

你可以在配置中设置：
- Ollama API地址与模型
- Aurora量化系统连接参数
- UI主题与字体

### 3. 启动系统
