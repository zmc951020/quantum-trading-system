
# 技能克隆架构设计

## 核心定位

**我们的系统已经很先进了！机器人不是来替换它的，而是作为增益层 —— 核心能力是「克隆Skills」：爬取、获取、吸收外部Skills，让我们的机器人获得系统本身不具备的能力。**

---

## 发现的宝藏项目

### 1. create-agent-skill (关键！)

**项目**: npmjs.com/package/create-agent-skill

**核心功能**: 将任意GitHub仓库转换成agent skill

**工作流程**:
```
1. 克隆/读取仓库
2. 检测项目类型 (Node.js/Rust/Go/Python)
3. 提取文档 (Markdown/MDX)
4. 提取类型 (TypeScript .d.ts)
5. 提取示例 (examples/demo/samples目录)
6. 生成SKILL.md清单
7. 安装到.agents/skills/目录
8. 为各个agent创建符号链接
```

**支持30个agents**: 包括Cline、Continue、Trae等

## 我们的机器人架构 (重点：技能克隆)

### 🎯 设计原则

1. **现有系统第一**: 我们的策略、优化器、回测、风控已经很先进了！保持原样
2. **机器人作为增益层**: 只做现有系统没有的功能
3. **核心能力：克隆Skills**: 爬取、获取、吸收外部Skills
4. **可插拔**: 随时可以启用/禁用某个Skill

### 🔧 核心模块设计

#### 模块1: SkillCloner (技能克隆器)

**功能**: 从GitHub/仓库克隆Skills

**流程**:
```
用户说 "克隆XXX技能"
  ↓
1. 发现 (crawl): 在GitHub/市场搜索相关Skills
2. 拉取 (fetch): clone仓库或下载
3. 分析 (analyze): 用ghrepo-mcp分析代码结构
4. 转换 (transform): 用create-agent-skill转换成我们的格式
5. 验证 (verify): 安全扫描 + 测试运行
6. 安装 (install): 放入.qs_robot/skills/目录
7. 注册 (register): 添加到技能索引
```

**数据源**:
- GitHub (直接仓库)
- Claude Plugin Hub
- 其他技能市场

---

#### 模块2: SkillMarketplaceManager (技能市场管理器)

**功能**: 发现、管理、评分Skills

**界面命令示例**:
```
/skills list                    # 列出已安装技能
/skills search "回测"          # 搜索技能
/skills clone <github-url>     # 克隆技能
/skills install <skill-name>   # 安装技能
/skills disable <skill-name>   # 禁用技能
/skills rate <skill-name>      # 评分技能
```

---

#### 模块3: SkillExecutor (技能执行引擎)

**功能**: 根据需求选择/组合Skills执行

**特点**:
- 按需加载 (减少token消耗)
- 技能组合 (多个Skills协作)
- 降级方案 (Skill失败时用备选方案)
